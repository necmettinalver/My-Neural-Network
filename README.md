## Kurulum

Projenin çalıştırılması için aşağıdaki gereksinimlere ihtiyaç vardır:
- Python 3.x
- Gerekli kütüphaneler (numpy, pandas, vs.)

Gerekli kütüphaneleri yüklemek için aşağıdaki komutları kullanabilirsiniz:

\`\`\`bash
pip install numpy
pip install pandas
\`\`\`
